# Awesome Shell Scripts

Easy install and administrate your Debian / Ubuntu applications and services

---

# Installation

## Download project
````shell
git clone https://github.com/xenetis/scripts
cd scripts
````
## Install (ex: opensnitch)
````shell
make install opensnitch
````
